# SiemensJava2026


# Solution:
I have a created a backend Springboot app that does the requiered busines logic. I have used Mailtrap to send the emails.
I have created a html for the view part. I use the Model-View-Controller pattern to show the data in view, controller to handle the interaction between view and model  and model to represent the data (there are also DTO for the JSON component communication). There is also a service layer that takes care of the data handling to leave the controller in a smalller size.

## Prerequisites
Install the needed dependencies/springboot
Used Java version 23
